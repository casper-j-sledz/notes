>> ### Span
>>> ```csharp
>>> const int indexNotFound = -1;
>>> 
>>> public static string StringSpanProcessing(this string text, string separator = "\n", string connector = " ")
>>> {
>>>   var stringBuilder  = new StringBuilder(); //Spans can't be stored as list, StringBuilder is the best way to construct string using spans.
>>>   var spanToProcess = text.AsSpan();
>>> 
>>>   while(!spanToProcess.IsEmpty)
>>>   {
>>>     var nextSeparatorIndex = spanToProcess.IndexOf(separator);
>>>     var spanTextPart = nextSeparatorIndex == indexNotFound ? spanToProcess : spanToProcess[..nextSeparatorIndex];
>>>     stringBuilder.Append(spanTextPart.Trim()).Append(connector);
>>>     spanToProcess = spanToProcess[(nextSeparatorIndex + separator.Length)..];
>>>   }
>>>   
>>>   return stringBuilder.ToString(0, stringBuilder.Length - connector.Length) ;
>>> }
>> ### Stream copy  & Buffers
>>> | Stream type               | Optimal Buffers
>>> | ------------------------- | ----------------------
>>> | File I/O (SSD/HDD)        | **32 KB – 128 KB** * ~85KB+ LOH (Large Object Heap)
>>> | Network streams           | **8 KB – 64 KB**
>>> | Memory streams            | **4 KB – 16 KB**
>>> | High-throughput file copy | **64 KB – 1 MB**
>>> ```csharp
>>> public static async Task<Stream> Copy(this Stream stream, CancellationToken cancellationToken)
>>> {
>>>   var streamCopy = new MemoryStream();
>>>   await stream.CopyToAsync(streamCopy, 81920, cancellationToken); // 81920 = 80 KB avoids LOH allocations
>>>   streamCopy.Position = 0; // Next read stream requires position reset
>>>   stream.Position = 0;
>>>   return streamCopy;
>>> }
